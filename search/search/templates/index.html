Document Search REST API

The Document Search REST API allows the user to search and find relevant documents based on a given query.

Available methods
GET /search Search for relevant documents
Search for relevant documents
GET /search

Route information

Query string parameters
text Type: String Title: Query Text Optional: false The text for which we wish to retrieve documents.
sources Type: String Title: Document Sources Optional: true The comma separated source names of the documents. When provided, returns only documents that are associated with any of the provided sources. Possible options: eurlex, ecolex.
Default value is eurlex.
locations Type: String Title: Associated Locations Optional: true The comma separated location names. When present, returns only the documents that are associated with any of the locations.
Default value is Null.
languages Type: String Title: Languages Optional: true The comma separated languages in English. When present, returns only the documents that are available in any of the provided languages.
Example: languages=english,german
informea Type: String Title: InforMEA Optional: true The comma separated InforMEA terms. When present, returns only the documents that are associated in any of the provided InforMEA terms.
Example: languages=english,german
limit Type: Integer Title: Limit Optional: true The number of retrieved documents.
page Type: Integer Title: Page Optional: true The page of the retrieved documents.
Response body attributes
documents Type:Object[] Title:Document Metadata Read only: true Metadata of the retrieved documents.
documents item properties
score Type: Number Title: Score Read only: true The relevance score of the document to the query.
document_id Type: Number Title: Document ID Read only: true The document ID.
title Type: String Title: Document Title Read only: true The document title (if present, otherwise None).
abstract Type: String Title: Document Abstract Read only: true The document abstract (if present, otherwise None).
link Type: String Title: Document Link Read only: true The link to the associated document or webpage containing more information.
date Type: Date Title: Document Date Read only: true The date of the document (if present, otherwise None).
celex Type: String Title: Document CELEX ID Read only: true The document CELEX ID (if present, otherwise None).
keywords Type: String[] Title: Document Keywords Read only: true The document keywords array (if present, otherwise None).
source Type: String Title: Document Source Read only: true The document source.
informea Type: String[] Title: Document InforMEA Terms Read only: true The document InforMEA terms array (if present, otherwise None).
languages Type: String[] Title: Document Languages Read only: true The array of languages the document is available in (if present, otherwise None).
subjects Type: String[] Title: Document Subjects Read only: true The array of subjects associated with the document (if present, otherwise None).
areas Type: String[] Title: Document Areas Read only: true The areas the document is associated with (if present, otherwise None).
query Type:Object Title:Query Parameters Read only: true The query parameters used to perform the search.
metadata Type:Object Title:Search Metadata Read only: true The search metadata used to support navigating through the results.
metadata item properties
total_hits Type: Number Title: Total Hits Read only: true The number of all relevant documents.
total_pages Type: Number Title: Total Pages Read only: true The maximum number of result pages.
prev_page Type: String Title: Previous Page Read only: true The URL to the previous search result page (if present, otherwise None).
next_page Type: String Title: Next Page Read only: true The URL to the next search result page (if present, otherwise None).
